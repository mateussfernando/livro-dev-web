# Livro.dev 📚💻

## 🚀 Sobre o Projeto
O **Livro.dev** é um site que recomenda livros para desenvolvedores, organizados por categorias como Front-end, Back-end, Cybersegurança, Inteligência Artificial, UX/UI e muito mais. A ideia é ajudar você a encontrar as melhores leituras para aprimorar suas habilidades na área de tecnologia.

## 📚 Categorias
